# ONI Colony Designer

**Oxygen Not Included** 向けのブラウザベースのコロニー設計ツールです。

ゲーム内で実際に建築を行う前に、建物配置や大型施設のレイアウトを事前に設計できます。

---

## このツールを作った理由

多くのONIプレイヤーは現在、

* スクリーンショット
* スプレッドシート
* お絵描きツール
* Sandboxセーブデータ

などを利用してコロニー設計を行っています。

ONI Colony Designer は、Oxygen Not Included 専用の設計環境を提供することを目的としています。

より簡単に、より正確に、より効率よくコロニー計画を立てられることを目指しています。

---

## 主な機能

### 建築物配置

* タイルグリッド上への建築物配置
* 複数サイズ建築物対応
* 建築物サイズ対応
* カテゴリ別表示
* 建築物検索・フィルタリング

### コロニー設計

* 拠点レイアウト設計
* 工業区画設計
* 発電設備設計
* ロケット施設設計
* 拡張計画設計

### セーブ・ロード

* レイアウト保存
* レイアウト読込
* Undo機能
* 後方互換性の維持

### 多言語対応

* 英語UI（デフォルト）
* 日本語UI
* 建築物名翻訳
* カテゴリ翻訳
* UI全体翻訳

---

## 建築物データベース

現在のデータベースは以下の特徴を持っています。

* バニラ版建築物216件収録
* Cargo由来データベース採用
* 建築物サイズ検証済み
* 建築物配置サイズ対応
* 英語名・日本語名対応

### 修正された代表的な建築物サイズ

データベース移行時に複数のサイズ誤りを修正しました。

| 建築物                 | 旧サイズ | 修正後       |
| ------------------- | ---- | --------- |
| Liquid Reservoir    | 3×3  | 2×3       |
| Transit Tube Access | 2×3  | 3×2       |
| Wood Burner         | 2×3  | 2×2       |
| Printing Pod        | 4×4  | 4×4（特別処理） |

## 既知の制限

本プロジェクトは現在MVP段階です。

### 部屋認識

部屋認識機能は現在試験実装です。

主要な部屋は認識可能ですが、一部の部屋条件はゲーム内挙動と完全一致していません。

正式な部屋ルール対応はMVP公開後に実施予定です。

### 建築物グラフィック

建築物アイコンはまだ実装されていません。

現在はレイアウト設計を優先するため、シンプルなアウトライン表示を採用しています。

### DLC対応

現在の公開版はバニラ版ONIを対象としています。

DLC対応は今後のアップデートで実装予定です。

## 開発方針

### Accuracy First

可能な限り実ゲームの挙動を優先します。

建築物サイズや設置条件は実際のONIに合わせることを目標としています。

### English First

内部データは英語を基準として管理します。

日本語は翻訳レイヤーによって提供されます。

### Save Compatibility

既存のセーブデータとの互換性維持を重視します。

### Practical Planning Tool

このプロジェクトはONI本体を再現することが目的ではありません。

コロニー設計を効率化することが目的です。
